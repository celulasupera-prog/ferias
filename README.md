# App de Aviso e Recibo de Férias

Aplicação web simples (HTML/CSS/JS) para gerar aviso e recibo de férias em um layout semelhante ao modelo enviado.

## Como usar

1. Abra `index.html` no navegador.
2. Preencha os dados no painel à esquerda.
3. Clique em **Imprimir / Salvar PDF** para exportar.

## Funcionalidades

- Atualização em tempo real dos campos.
- Cálculo automático de:
  - total base;
  - férias;
  - 1/3 de férias;
  - descontos de INSS e IRRF por percentual;
  - total líquido.
- Modo de impressão otimizado.

## Observação

Os cálculos são auxiliares e podem precisar de ajuste conforme regra contábil/folha da empresa.
